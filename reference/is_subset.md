# Check if one set is a subset of another

Determines if all elements in set `x` are also present in set `y`.

## Usage

``` r
is_subset(x, y)
```

## Arguments

- x:

  A numeric, character, or logical vector.

- y:

  A numeric, character, or logical vector.

## Value

A logical value indicating whether `x` is a subset of `y`.
