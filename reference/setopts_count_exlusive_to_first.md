# Count of Elements Exclusive to First Set

Counts the number of elements that are in the first set but not in the
second set.

## Usage

``` r
setopts_count_exlusive_to_first(x, y)
```

## Arguments

- x:

  A vector of elements.

- y:

  A vector of elements.

## Value

A scalar representing the number of elements that are in the first set
but not in the second set.
