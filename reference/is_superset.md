# Check if one set is a superset of another

Determines if all elements in set `y` are also present in set `x`.

## Usage

``` r
is_superset(x, y)
```

## Arguments

- x:

  A numeric, character, or logical vector.

- y:

  A numeric, character, or logical vector.

## Value

A logical value indicating whether `x` is a superset of `y`.
