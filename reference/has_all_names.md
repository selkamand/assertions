# Check if a named object has all specified names

This function returns a logical value indicating whether the object `x`
has all the names specified in `names`.

## Usage

``` r
has_all_names(x, names)
```

## Arguments

- x:

  a named object

- names:

  A character vector of names to check for in `x`.

## Value

A logical value indicating whether `x` has all the names specified in
`names`
